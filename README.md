# 🚀 StartWise AI — AI Co-Founder for Indian Startups

StartWise AI is a premium AI-powered co-founder platform that helps first-time Indian founders transform startup ideas into investor-ready businesses through AI-driven market analysis, execution roadmaps, AI VC simulations, investor discovery, and intelligent startup guidance.

---

# 🌟 Overview

StartWise AI acts as a digital AI co-founder that assists aspiring entrepreneurs in validating startup ideas, understanding market opportunities, building structured execution roadmaps, improving investor readiness, and connecting with startup ecosystems.

The platform combines:
- AI-powered startup intelligence
- Voice-to-voice AI VC simulations
- Pitch deck generation
- Investor discovery
- AI outreach automation
- Interactive SaaS dashboards

All packed inside a futuristic, premium AI SaaS experience.

---

# 🚨 Problem Statement

Early-stage founders often struggle with validating startup ideas, understanding market demand, building investor-ready pitch decks, finding relevant investors, and executing their ideas in a structured manner due to lack of mentorship, startup knowledge, and access to professional guidance.

---

# 💡 Proposed Solution

StartWise AI acts as an intelligent AI co-founder that helps founders analyze startup ideas, generate daily execution roadmaps, simulate realistic AI VC interactions, create investor-ready pitch decks, discover nearby investors, and draft professional fundraising emails — all through a premium AI SaaS platform designed for startup growth and investor readiness.

# ⚙️ Setup & Local Development

## Prerequisites
- Node.js
- Gemini API Key
- Sarvam API Key
- PostgreSQL Database URI

---

## Installation

```bash
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key
SARVAM_API_KEY=your_api_key
DATABASE_URL=your_postgresql_url
```
